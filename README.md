AwesomeFrame是一款聚集了目前安卓开发中常用的流行UI框架的应用

目前已完成的框架：  
* Card
	- floding-cell-android
	- SwipeStack（待改进）
* Dialog
	- RxTool（待完成）
	- sweet-alert-dialog（第一项待改进）
* Pick
	- Android-PickerView
	- CityPicker  
* Refresh
	- SmartRefreshLayout  
* Banner
	- banner
	- BGABanner-Android
* About
	- about-page
	- material-about-library
	- AboutLibraries
